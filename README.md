# Flappy Kids 🐦

A fun Flappy Bird-style browser game for kids, built with React and HTML5 Canvas.

**Live at:** https://flappy-kids.vercel.app

## Features

- Choose your character face from 9 custom photos
- 5 colorful themes: Sunny Sky, Night Sky, Rainbow Land, Snow World, Cloud World
- Collectible golden stars and balloon power-ups
- Unlockable hats, crown, and sunglasses based on score
- Sparkle trail behind the character
- Sound effects for jump, score, and game over
- Confetti celebration at score 10
- Top 5 local leaderboard saved in the browser
- Save your score as a screenshot
- Works on mobile and desktop

## Tech Stack

- React 18
- HTML5 Canvas
- Web Audio API
- Vite
